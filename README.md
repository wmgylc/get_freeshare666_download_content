# 实用油猴脚本集合

包含两个独立的实用油猴脚本：下载按钮自动点击器和百度网盘密码自动填入助手。

## 脚本列表

### 1. 下载按钮点击器 (`simple-download-clicker.user.js`)
- 🎯 **智能识别**：自动识别页面中的"立即下载"按钮
- 🚀 **一键下载**：点击页面底部的"开始下载"按钮，自动点击所有下载链接
- 📱 **响应式设计**：按钮样式适配不同屏幕尺寸
- 🔄 **实时监控**：持续监控页面变化，自动处理新出现的下载按钮
- 🌐 **支持网站**：freeshare666.com 及其相关页面

### 2. 百度网盘密码助手 (`baidu-pan-password-helper.user.js`)
- 📋 **剪贴板监控**：实时监控剪贴板内容，自动识别提取码
- 🔐 **智能填入**：检测到有效提取码时自动填入密码框
- ⚡ **自动提交**：填入密码后自动点击"提取文件"按钮
- 💾 **自动保存**：验证成功后自动点击"保存到网盘"按钮
- 🔔 **通知提醒**：操作完成后显示桌面通知和页面提示
- 🎯 **双重检测**：结合定时检测和页面变化监听，确保不遗漏任何机会
- 🌐 **支持网站**：pan.baidu.com 分享页面

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 根据需要选择安装脚本：
   - [下载按钮点击器](./simple-download-clicker.user.js)
   - [百度网盘密码助手](./baidu-pan-password-helper.user.js)
3. 在 Tampermonkey 管理面板中确认安装

## 使用说明

### 下载按钮点击器
1. 访问支持的下载网站（如 freeshare666.com）
2. 页面底部会出现"开始下载"按钮
3. 点击按钮即可自动点击页面中的所有下载链接

### 百度网盘密码助手
1. 复制百度网盘的提取码到剪贴板
2. 访问百度网盘分享链接
3. 脚本会自动检测提取码并填入密码框
4. 自动点击"提取文件"按钮
5. 验证成功后自动点击"保存到网盘"按钮
